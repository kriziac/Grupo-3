# Art Snap

Se trata de un concurso de fotografía creativa, diseñado para empoderar a los adolescentes. A través de una página web, donde se les brinda la oportunidad de expresar su creatividad a adolescentes de 10 a 17 años para explorar la riqueza cultural de la ciudad.

# Configuración del entorno

Este proyecto consta de dos partes: el backend desarrollado en Java con Spring y una base de datos MySQL, y el frontend desarrollado en tecnologías web estándar como HTML, JavaScript y CSS.

### Despliegue del Frontend

Ejecutar el servidor frontend y abrir el archivo "index.html" en tu navegador o abrir la IP publica.
